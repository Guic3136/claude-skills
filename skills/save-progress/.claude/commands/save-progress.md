---
description: 保存当前会话进度到带时间戳的文件
---

请保存当前会话的进度摘要。执行以下操作：

1. 检查当前工作目录下是否存在 summaries 目录，如果不存在则创建它
2. 生成带时间戳的文件名：session-summary-YYYY-MM-DD-HHMM.md（例如 session-summary-2026-03-17-1430.md）
3. 在文件中记录以下内容：
   - 当前任务目标
   - 已完成的工作
   - 关键决策和发现
   - 下一步计划

4. 将文件保存到 summaries/ 目录下
5. 向用户确认文件已创建，并显示文件路径
